# Applix

Site destinado a Applix, um dos mais podemoros sistema de gestão para empresas. Esse sistema possui diversas funcionalidades, como gestão financeira, faturamento, estoque, compras e recebimento, vendas, serviços, projetos, crm, controle de produção, planejamento financeiro, gestão de pessoas, atendimento ao cliente e até cálculo de comissões.

Além disso, possui um blog com artigos postados semanalmente. Nesse site institucional foi desenvolvido o layout do blog, na qual puxa do seu banco de dados todos os artigos e exibe no site, foi adicionado campos de filtragem de blocos e um dropdown na versão de celular para facilitar o a experiência do usuário.

## Funcionalidades

- Layout dinâmico dos artigos do blog.
- Filtragem dos artigos.
